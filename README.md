The model classifies images into the following categories:
- normal
- scratch

## CNN Concepts

### What is Convolution?
Convolution helps extract important features from images such as edges, patterns, and textures.

### Why is Pooling Used?
Pooling reduces image dimensions and computational complexity while preserving key information.

### Why is ReLU Used?
ReLU improves training speed and helps CNNs learn complex patterns effectively.

### Why are CNNs Better for Images?
CNNs automatically detect spatial image features and are more efficient than regular neural networks for image-related tasks.
